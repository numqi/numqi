# channel

shape conventions for quantum channels

1. Kraus operator `(rank, dim_out, dim_in)`
2. Choi operator `(dim_in*dim_out, dim_in*dim_out)`
3. Super operator `(dim_out*dim_out, dim_in*dim_in)`
